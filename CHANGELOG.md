# Changelog

## [1.0.0]

- Initial release of **Kama Dark**.
- Warm brown/cream/gold palette derived from the Kama web app.
- Low-contrast editor text (~8.5:1) for reduced eye strain.
- Borderless, flat layout with three-tier tonal depth (frame → sidebar → editor).
- Syntax highlighting, terminal ANSI colors, bracket-pair colors, and git decorations.
